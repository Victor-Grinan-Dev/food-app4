# FoodApp:

Food app is a class project for the react22k course.

Source Code: Github
Hosted in: Netlify
## Live site:
https://food-app4.netlify.app/

### Dependencies:

- React & React Router Dom
- Axios
- Firebase
- React Device Detect 

### Installation:
clone the repo, cd in it and type in terminal:

```shell
npm install
npm start
```
